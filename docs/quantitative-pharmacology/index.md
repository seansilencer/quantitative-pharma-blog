# 定量药理建模

> 通过数学建模与仿真手段，对药物的 PK/PD 过程进行量化分析

## 核心方法

### PBPK / 生理药代动力学
基于生理学的药代动力学模型，通过整合生理参数（血流量、组织体积等）与药物特异性参数，预测药物在全身各组织中的浓度随时间变化。

**应用场景：**
- 儿科药物研发（由成人数据外推）
- 药物相互作用（DDI）预测
- 特殊人群（肝/肾功能损害）剂量调整
- 虚拟生物等效性（IVBE）评估

### PopPK / 群体药代动力学
将药代动力学数据汇总分析，同时估计典型人群参数和个体变异来源（残差、间变异、个体内变异）。

**应用场景：**
- 稀疏采样数据解析
- 临床试验剂量优化
- 种族差异评估
- 贝伐珠单抗等大分子药物的剂量预测

### PK/PD 关联分析
链接药物暴露量（PK）与药效响应（PD），建立定量关系模型，支持给药方案优化。

**应用场景：**
- 抗肿瘤药物的疗效/毒性预测
- 抗菌药物的 PK/PD 靶值优化
- 罕见病药物的剂量选择
