# PopPK 群体药代动力学

## 概念

群体药代动力学（Population PK, PopPK）利用稀疏临床采样数据，同时估计：
- **典型值**（THETA）：人群平均参数
- **个体变异**（ETA）：个体间随机效应
- **残差变异**（EPS）：测量/模型误差

## 核心方法

- **非线性混合效应模型**（NONMEM, PsN, MONOLIX）
- **贝叶斯估计**（BGM, Stan）
- **机器学习辅助**（Random Effects建模）

## 关键指标

| 指标 | 含义 | 监管要求 |
|------|------|---------|
| CL | 清除率 | FDA/EMA 必报 |
| Vd | 表观分布容积 | FDA/EMA 必报 |
| ω² | 个体间变异 | 通常报告 |
| σ² | 残差变异 | 通常报告 |

## 应用场景

- 儿科外推（瀑布式开发）
- 剂量个体化（TDM 结合 PopPK）
- 临床试验模拟
