# 特殊人群药理

## 肝功能损害患者

### 分级标准（Child-Pugh）

| 评分 | 等级 | 手术风险 | 描述 |
|------|------|----------|------|
| 5-6 | A（轻度） | Minimal | 代偿良好 |
| 7-9 | B（中度） | Moderate | 显著功能损害 |
| 10-15 | C（重度） | High | 失代偿 |

### 剂量调整策略

- **FDA 指南**：基于 Child-Pugh 评分的 PK 研究
- **定量方法**：PBPK 模型预测中/重度肝损 PK 变化
- **关键参数**：肝脏代谢（CYPs）、胆汁排泄、转运体

## 肾功能损害患者

### 分期（eGFR）

| 分期 | eGFR (mL/min/1.73m²) | 描述 |
|------|----------------------|------|
| G1 | ≥90 | 正常或高 |
| G2 | 60-89 | 轻度下降 |
| G3a | 45-59 | 轻度至中度 |
| G3b | 30-44 | 中度至重度 |
| G4 | 15-29 | 重度 |
| G5 | <15 | 肾衰竭 |

### 调整原则

- 经肾排泄药物：剂量 = 常规剂量 × 肾脏排泄比例
- 透析影响：考虑透析清除率和补充剂量
- PBPK + 透析建模

```
成人数据 ──→ 异速缩放 ──→ 青少年 ──→ 儿童 ──→ 婴幼儿
（PK/PD）  （ALLometric）  (12-17y)  (2-11y)  (<2y)
```

### 关键技术

- **异速缩放**：基于体重的 CL/Vd 幂律关系
- **儿科制剂：PK-Sim +YM）
- ** PopPK + 贝叶斯**：稀疏采样优化

## 老年人

### 老年 PK/PD 改变

- **PK**：肝/肾功能自然衰退、多药合用（DDI 风险↑）
- **PD**：受体敏感性改变、homeostatic reserve 下降
- **认知/依从性**：简化给药方案的重要性

### ICH E7 建议

≥65 岁受试者须纳入关键临床试验，必要时专门开展老年人群研究。
