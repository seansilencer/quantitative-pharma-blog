# 仿制药与生物等效性（BE）

## 生物等效性（BE）核心概念

生物等效性指在相同试验条件下，试验制剂与参比制剂的吸收速度和程度无显著性差异。

**统计标准**：BE 界值通常为 80%~125%（对数转换后）

$$0.80 \leq \frac{T}{R} \leq 1.25$$

## 关键研究设计

| 类型 | 适用场景 | 样本量 |
|------|----------|--------|
| 2×2 交叉 | 普通口服固体制剂 | 24~36 例 |
| 部分重复 | 高变异药物（CV>30%） | 视变异度 |
| 全重复 | 窄治疗窗药物 | 更大 |
| 食物效应 | 口服制剂 | 独立 BE 研究 |

## 高变异药物策略

- **比例标化平均法**（Scaled Average Bioequivalence）
- **参比制剂校正**（Reference-scaled）
- 主体 Bootstrap 评估

## 复杂制剂 BE

- **缓控释制剂**：多剂量稳态研究，食物效应研究
- **吸入制剂**：气雾剂/粉雾剂的 PK/PD 等效性
- **透皮制剂**：皮肤药代动力学（MDPK）
- **局部作用 GI 药物**：BE 豁免条件（BCS I 类）

## 指导原则

- FDA: ANDA 申报要求（Code of Federal Regulation 320）
- NMPA: 《化学药品生物等效性研究技术指导原则》
- ICH M13A: Bioequivalence for immediate-release solid oral dosage forms
